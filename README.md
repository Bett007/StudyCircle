# StudyCircle

#### StudyCircle is a collaborative productivity platform designed to help students study smarter and track their academic progress in a shared and supportive environment.

## Features

### 1. Sprint Room
#### A focused real-time study space with:
- Pomodoro Timer for deep work
- Spotify Integration to maintain study flow
- Chatbox for team discussions and check-ins
- Personal & Group Sprint Modes for flexible study sessions

### 2. Progress Feed
#### A shared activity feed where members can:
- Post study updates
- Share feedback


### 3. SOS 
#### A dedicated help section that allows students to:
- Request support when stuck
- Receive help from students

### 4. Measurement Hooks
#### Tracking tools that measure:
- Productivity patterns
- Self and Mood Check-Ins

## Technology Used
1. Frontend: React + Vite
2. Hosting: Vercel
3. Styling: Custom CSS 
4. API: Spotify, Firebase

## Project Setup
 1. Clone the Repository
`git clone https://github.com/<your-repo>/studycircle.git`

2. Install Dependencies
`npm install`

3. Start Development Server
`npm run dev`

4.Build for Production
`npm run build`
